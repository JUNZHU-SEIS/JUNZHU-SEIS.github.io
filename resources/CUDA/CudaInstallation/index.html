    Runfile installation

    1. download CUDA (Version=11.2)
    wget https://developer.download.nvidia.com/compute/cuda/11.2.0/local_installers/cuda_11.2.0_460.27.04_linux.run
    1. install it by run the following command
    sudo sh cuda_11.2.0_460.27.04_linux.run
    1. If any, just follow the instructions given by CUDA11.2.0 DOCUMENT and Zhihu blog (中文)

    test Numba

    1. install numba & cudatoolkit using conda
    conda install cudatoolkit
conda install numba
    1. test Numba
    from numba import cuda
def cpu_print():
    print("print by cpu.")
@cuda.jit
def gpu_print():
    print("print by gpu.")

def main():
    gpu_print[1, 2]()
    cuda.synchronize()
    cpu_print()

if __name__ == "__main__":
    main()
